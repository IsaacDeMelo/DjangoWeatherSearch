ⓘ Este projeto foi feito utilizando tecnologias como Django, HTML5, CSS3 e JS. Consiste num sistema de requisição, buscando informações sobre o clima de várias cidades do mundo em tempo real: Temperatura (Em C°), incluindo as margens de temperatura tanto máxima quanto mínima, assim como o nome do país onde a cidade está localizada. A versão da API é gratuita, permitindo 50 requisições por minuto.
